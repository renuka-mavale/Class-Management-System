## 🎓 Class Management System

A simple menu-driven Class Management System built using Java.
This console application allows users to manage Courses, Faculties, Batches, and Students through an interactive menu system.

## 🛠 Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collection Framework

## ▶ How to Run

### Using Eclipse
1. Open Eclipse IDE.
2. Import the project.
3. Open ControllerClass.java.
4. Click Run As → Java Application.
5. Use the menu to manage Courses, Faculties, Batches, and Students.

## 📋 Menu Options

1. Add Course
2. Display Course
3. Add Faculty
4. Display Faculty
5. Add Batch
6. Display Batch
7. Add Student
8. Display Student
9. Exit
